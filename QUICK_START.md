# VoxSplit - Quick Start

## What is This?

VoxSplit is a web application that separates voices and removes background noise from audio files using AI.

## Features

- Upload audio files (MP3, WAV, M4A, FLAC, OGG, AAC)
- Separate multiple speakers automatically
- Remove background noise
- Download cleaned audio files
- Free to use!
